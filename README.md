## 介绍

本项目实现了比较 MySQL 建表语句的差异，并生成结构同步代码，功能类似 Navicat 的结构同步。

**支持：**

* [x] 表选项（注释、引擎等）

* [x] 字段

* [x] 索引

* [x] 外键

* [x] 分区

## 安装

`composer require yurunsoft/sql-diff`
